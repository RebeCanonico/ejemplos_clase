Definir estílos de HTML con CSS

CSS (Cascade Style Sheets) se usa para controlar el formato visual de una pagina.

Con CSS se puede controlar el color, fuente, tamaño del texto, espaciado, alineamiento, imagenes de fondo, cómo se ve una página dependiendo del tamaño de pantalla del dispositivo que estamos usando, etc.

Formas de usar el CSS

  • Inline - Usando el atributo style dentro del elemento HTML.
  • Internal - Usando la etiquéta <style> dentro del elemento HTML.
  • External - Usando un archivo CSS externo.

External CSS

Las dos principales ventajas de externalizar el CSS son:

  • Los estilos s epueden aplicar a tantos documentos como queramos.
  • La longitud del código disminuye sensiblemente.
