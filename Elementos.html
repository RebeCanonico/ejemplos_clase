Links o hipervinculos

Los links se encuentran en todo tipo de paginas que permiten al usuario navegar por la web o por los documentos de la web.

Cuando clicas en un link te mueve a otro documento o a una parte concreta del mismo.

Cuando se pasa el cursor sobr eun link, éste se torna en una mano.

Un link se define mediante un elemento <a>

La sintaxis de un link, por norma general, es: <a href="url">Text del link</a>

El atributo mas importante de un link es el "href" ya que es este el que indica el punto al que nos llevará el link.

Los links tienen 3 estados:

  • Son azules cuando no han sido viditados.
  • Morados cuando ya han sido clicados.
  • y rojos cuando están activos.

Los links se pueden modificar mediante CSS para que tengan un aspecto completamente distinto.
