Formateo del texto de HTML

Elementos destinados especificamente a formatear el texto:

  • b - Texto en negrita
  • strong - Texto importante
  • i - Texto en cursiva
  • em - Texto énfasis
  • Mark - Texto marcado
  • small - Texto de menor tamaño
  • del - Texto tachado
  • ins - Texto insertado
  • sub - Texto subscrito
  • sup - Texto superescrito

Esto es un párrafo. El texto que queramos destacar visualmente lo haremos con negrita El texto importante irá con la etiqueta strong.
Mientras que la b es meramente visual, strong influye en el posicionamiento web de la pagina.

Elementos de citación

  • Blockquote - Define una sección citada de otra fuente.
  • q - Define una cita corta.
  • abbr - Define una abreviatura o acrónimo.
  • address - Define la información de contacto del autor del articulo.
  • cite - Define el título de un trabajo creativo. Como un poema, libro o cancion.
  • bdo - Se usa para definir la dirección del texto.

Estílos

Los estílos definen el aspecto visual de los elementos html que acompañan. La sintaxis es la siguiente:
<elemento style:="propiedad:valor;">

ejemplo

Este parrafo tiene propiedad de fondo y color asignadas.

EL lenguaje en el que se determinan los estilos es CSS.

Una manera más correcta de determiar los estílos es mediante la etiquéta style ubicada dentro de la etiqueta head.
